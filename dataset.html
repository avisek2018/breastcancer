The data set we have used here is Breast Cancer Data set from UCI Machine Learning Repository. This databases was obtained from the University of Wisconsin Hospitals, Madison from Dr. William H. Wolberg.

The data set can be accessed here


Here are the data set attributes details ....



Attribute Data Type Domain
Sample code number Numeric id number
Clump Thickness Numeric 1 - 10
Uniformity of Cell Size Numeric 1 - 10
Uniformity of Cell Shape Numeric 1 - 10
Marginal Adhesion Numeric 1 - 10
Single Epithelial Cell Size Numeric 1 - 10
Bare Nuclei Numeric 1 - 10
Bland Chromatin Numeric 1 - 10
Normal Nucleoli Numeric 1 - 10
Mitoses Numeric 1 - 10
Class: Numeric (2 for benign, 4 for malignant)

We have mapped 2 to 0 and 4 to 1 for our analysis.
